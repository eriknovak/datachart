    Charts Module

    datachart.charts

    Module containing the charts.

    The charts module contains the methods to create the plots and figures.

    Creates the bar chart.

    Examples:

    >>> from datachart.charts import BarChart
>>> figure = BarChart(
...     data=[
...         {"label": "cat1", "y": 5},
...         {"label": "cat2", "y": 10},
...         {"label": "cat3", "y": 15},
...         {"label": "cat4", "y": 20},
...         {"label": "cat5", "y": 25}
...     ],
...     title="Basic Bar Chart",
...     xlabel="LABEL",
...     ylabel="Y"
... )
    PARAMETER DESCRIPTION
    data
